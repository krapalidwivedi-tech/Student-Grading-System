# Student Grading System

## Project Overview

Student Grading System is a Python-based project that allows teachers to enter student details and marks. The system automatically calculates the total marks, percentage, and grade, then stores the data in an SQLite database.

## Features

* Enter student roll number and name
* Enter marks for 3 subjects
* Calculate total marks
* Calculate percentage
* Generate grade automatically using if-else conditions
* Store student records in SQLite database
* Search student records using Roll Number
* Uses SQL INSERT and SELECT queries with WHERE clause

## Technologies Used

* Python
* SQLite3
* SQL Queries

## Grading Criteria

| Percentage   | Grade |
| ------------ | ----- |
| 90 and above | A+    |
| 75 - 89      | A     |
| 60 - 74      | B     |
| 40 - 59      | C     |
| Below 40     | Fail  |

## Database Fields

* Roll Number
* Name
* Subject 1 Marks
* Subject 2 Marks
* Subject 3 Marks
* Total Marks
* Percentage
* Grade

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Run the Python file:

```bash
python code.py
```

4. Enter student details when prompted.
5. View calculated grade and save data to the database.
